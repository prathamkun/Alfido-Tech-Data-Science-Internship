# Zomato Dataset Analysis

## Objective

Analyze restaurant and review data to identify patterns in ratings, cuisines, locations, and pricing.

## Dataset

Zomato Restaurant Dataset

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* WordCloud
* Google Colab

## Data Cleaning

* Removed duplicate records
* Handled missing values
* Converted ratings to numeric format
* Cleaned cost column
* Removed invalid cuisine records

## Visualizations

1. Top 10 Restaurant Locations
2. Top 10 Most Popular Cuisines
3. Price vs Rating
4. Correlation Heatmap
5. Cuisine Word Cloud

## Key Findings

* BTM has the highest concentration of restaurants.
* North Indian cuisine is the most popular cuisine.
* Ratings are generally between 3.5 and 4.5.
* Price has a weak relationship with ratings.

## Recommendations

* Partner with highly rated restaurants in key locations.
* Focus on popular cuisine categories.
* Use ratings and reviews for personalized recommendations.
* Target mid-range restaurants for marketing campaigns.
* Create data-driven content for users.
